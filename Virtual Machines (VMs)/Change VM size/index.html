    1. Before resize

    1. ลองดูว่า VM ของเรา รันอยู่บน Hyper-V host ที่ชื่อว่าอะไร

    Get-ItemProperty -Path "HKLM:\SOFTWARE\Microsoft\Virtual Machine\Guest\Parameters" | Select-Object HostName

    2. ลองสร้าง file ทิ้งไว้ใน
